📸 PhotoGallery

A lightweight and responsive Photo Gallery Web Application built using HTML, CSS, and JavaScript, designed to showcase images in a clean grid layout with interactive UI behavior.

🚀 Live Overview
  This project renders a static image gallery where users can:
      
      Browse images in a grid layout
      Hover over images for visual effects
      View images in a larger preview (if implemented in JS)
      Experience responsive design across devices
      ✨ Features (Based on Your Implementation)
      📷 Grid-based image layout using CSS
      🎯 Hover effects on images (UI interaction focus)
      ⚡ Pure frontend — no backend dependency
      📱 Responsive design (mobile + desktop)
      🧩 Simple and modular file structure
      🛠️ Tech Stack
      HTML5 → Structure
      CSS3 → Styling, grid, hover effects
      JavaScript (Vanilla) → DOM interactions (if used)
